## Description

皮皮虾视频去水印，获取了原版视频地址

- Jun 3 2019 可用
- Nov 17 2019 可用
- April 20 2020 修复 no content without user-agent

## Usage

`brew install python`

`pip3 install requests` or `pip install requests`

`$ python run.py rR5Ppu`

The video will be downloaded to the same directory
